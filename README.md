# IconicAgents
Girlypop Icon's Agent Files

A collection of GitHub Copilot custom agents sourced from [girlypopicon/winframe](https://github.com/girlypopicon/winframe).

## Agents

| Agent | Description |
|-------|-------------|
| [API Architect](.github/agents/API-Architect.agent.md) | Designs and reviews REST APIs, GraphQL schemas, and service-to-service communication patterns with a focus on consistency and developer experience. |
| [Avalonia .NET Developer](.github/agents/AvaloniaNetDev.agent.md) | Expert Windows/desktop application development agent specializing in Avalonia UI, .NET, C#, and XAML best practices. |
| [Code Review Agent](.github/agents/Code-Review.agent.md) | Ruthless but constructive code reviewer focused on security, performance, maintainability, and correctness. |
| [Database Architect](.github/agents/Database-Architect.agent.md) | Designs schemas, writes optimized queries, and enforces data modeling best practices for SQL and NoSQL databases. |
| [DevOps Pipeline](.github/agents/DevOps-Pipeline.agent.md) | Designs CI/CD pipelines, Dockerfiles, Kubernetes manifests, and infrastructure-as-code with security and reliability baked in. |
| [Documentation Writer](.github/agents/Document-Writer.agent.md) | Writes clear, concise technical documentation including README files, API docs, inline code comments, and architecture decision records. |
| [Git Workflow](.github/agents/Git-Workflow.agent.md) | Manages branching strategy, commit conventions, PR templates, and release management for teams. |
| [Refactoring Expert](.github/agents/Refactoring-Expert.agent.md) | Restructures existing code for better readability, maintainability, and performance without changing external behavior. |
| [Security Hardener](.github/agents/Security-Hardener.agent.md) | Identifies vulnerabilities, enforces secure coding practices, and reviews code for OWASP Top 10 and common security weaknesses. |
| [Test Engineer](.github/agents/Test-Engineer.agent.md) | Writes thorough, well-structured tests with focus on coverage, edge cases, and testing best practices across all frameworks. |
| [WinDev](.github/agents/WindowsDev.agent.md) | Expert Windows/desktop application development agent specializing in Avalonia UI, .NET, C#, and XAML best practices. |
